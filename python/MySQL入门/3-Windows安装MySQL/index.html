一、下载

第一步：打开网址，https://www.mysql.com ，点击downloads之后跳转到https://www.mysql.com/downloads

第二步 ：跳转至网址https://dev.mysql.com/downloads/ ，拉到最下方，下载社区版

第三步 ：进入地址：https://dev.mysql.com/downloads/

第四步：https://dev.mysql.com/downloads/mysql/，这里我们看到的是最新版本，点击Archives，选择windows版，5.7版本

第五步：https://downloads.mysql.com/archives/community/，选择平台版本下载

二、解压

下载的zip文件解压，将解压之后的文件夹放到任意目录下，这个目录就是mysql的安装目录。

三、配置

打开mysql目录，新建文件为my.ini或者my.cnf

3.3 安装MySQL服务

以管理员身份打开cmd窗口后，将目录切换到你解压文件的bin目录，输入mysqld install回车运行

四、环境变量

在系统变量PATH后面添加: 你的mysql bin文件夹的路径（如C:\Program Files\mysql-5.7.30-winx64\bin）

这样在任意位置敲击 mysql，都能找到该命令

六、启动mysql服务

以管理员身份在cmd中输入：net start mysql

服务启动成功之后，就可以登录了，输入mysql -u root -p（密码在data目录下xx.err文件中）

6.1 停止mysql

net stop mysql：停止mysql

注意：在windows操作系统上没有重启mysql服务的命令，如果要重启服务，只能先stop再start。
