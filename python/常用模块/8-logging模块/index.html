1.3 v3

logging模块包含四种角色：logger、Filter、Formatter对象、Handler

  1. logger：产生日志的对象
  2. Filter：过滤日志的对象
  3. Formatter对象：可以定制不同的日志格式对象，然后绑定给不同的Handler对象使用，以此来控制不同的Handler的日志格式
  4. Handler：接收日志然后控制打印到不同的地方，FileHandler用来打印到文件中，StreamHandler用来打印到终端
